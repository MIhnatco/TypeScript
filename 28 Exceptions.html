Handling Exceptions in TypeScript.

  • Exceptions are unexpected or erroneous situations that can occur during the execution of a program.
  • They disrupt the normal flow of code execution and must be handled to prevent crashes and provide graceful error reporting.

Using 'try' and 'catch'

  • In TypeScript, you can handle exceptions using 'try' and 'catch' blocks.
  • Code that might raise an exceptions is placed within a 'try' block.
  • If an exception occurs, it's caught and handled in the corresponding 'catch' block.

Syntax

    
        try {
            //code that might throw an exception
        } catch (error){
            //code to handle the exception
        }

Example:

    
        try {
            const result 10 / 0;   //This will throw a division by zero exception
            console.log(result);   //This won't be executed
        } catch (error){
            console.error(`An error occurred: ${error.message}`)
        }

Throwing Custom exceptions:

  • You can throw custom exceptions using the 'throw' statement.
  • This allows you to create specific error messages and types for your application.

Syntax:

    
        throw new Error("Custom error message")

Example:

    
        function divide(x: number, y: number): number {
            if(y===0){
                throw new Error("Division by zero is not allowed.")
            }

            return x / y;
        }


        try {
            const result = divide(10,0);
            console.log(result)
        } catch(error){
            console.error(`An error occurred: ${error.message}`)
        }

Using "finally":

  • You can also use a 'finally' block to execute code regardless of whether an exception is thrown or not.

Syntax:

    
        try {
            // Code that might throw an exception
        } catch (error) {
            // Code to handle the exception
        } finally {
            // Code that always executes
        }

Example:

    

        try {
            console.log("Inside try block");
            throw new Error("An exception occurred.")
        } catch (error) {
            console.error(`An error occurred: ${error.message}`)
        } finally {
            console.log("This code always executes.")
        }
