Tuples

Tuples are ordered collections of elements with a fixed number of elements, where each element can have its own specified data type. Unlike arrays, tuples allow you to define the type of each element in a structured manner.

Creating Tuples and specifying element types:

You can create tuples by specifying the types for each element within parentheses. The number of specified types must match the number of elements in the tuple. 

    
        const person: [string, number, boolean] = ["Alice", 30, true]

In this example, 'person' is a tuple with three elements, and their types are specified.

Accessing Tuple elements:

You can access tuple elements using square brackets and zero-based indices: 

    
        const name: string = person[0];
        const age: number = person[1];
        const isStudent: boolean = person[2]

Modifying Tuples:

You can modify tuple elements directly by assigning new values to them, as long as the new values match the specified types. 

    
        person[1] = 31;  //modifying the age

Use cases for Tuples:

Tuples are useful for scenarios where you need to maintain a structured collection of elements with specific types. Some common use cases include representing pairs of triplets of values, such as coordinates, database records, or function return types.

We can return a tuple from a function: 

    
        function coord(): [number, number]{
            return [3,3]
        }
  • 'function coord(): [number, number]': this function is explicitly annotated to return a tuple with two elements, both of type 'number'. This means the function is expected to return an array-like structure with exactly two numbers.
  • 'return [3,5]' the function returns a tuple containing two numbers, 3 and 5.

You can call this function to get the coordinates: 

    
        const coordinates: [number, number] = coord();
        console.log(coordinates); // Output: [3,5]

In this example, the coord function is called, and the returned tuple is assigned to the 'coordintes' variable.

This demonstrates how yu can use tuples in TypeScript to represent structured data, such as coodinates, where you have specific types for each element. Tuples are useful for maintainig type safety and clear data structure definitions.

You can create an array of tuples in TypeScript by specifying the tuple types for each element within the array.
Here is an example of an array of tuples: 

    
        const people: [string, number][] = [
            ["Alice", 30],
            ["Bob", 25],
            ["Charlie", 35]
        ]

In this example:

  • 'people' is an array of tuples where each tuple has two elements.
  • The first element in each tuple is of type 'string' and the second element is of type number.

Mnemonic Devices:

  • Think of tuples as "typed arrays" with a predefined structure.
  • Use parentheses to remember tuple creation, and square brackets to access elements.
