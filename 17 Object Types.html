Object Types in TypeScript

An object is a data structure in TypeScript used to store related data and methods. It allows you to group together properties and functions that are related to a specific entity, concept, or thing.

Creating Objects Using Object Literals:

You can create objects using object literals, which is a straightforward way to define an object and its properties. Properties are defined as key-value pairs within curly braces. 

    
        let person = {
            name: "Alice",
            age: 30,
            greet: function() {
              console.log("Hello!");
            }
          };

Object-Oriented Programming (OOP) Concepts:

Objects are a fundamental concept in object-oriented programming. Understanding objects, classes, and instances is crucial for building complex systems.

Mnemonic Devices:

  • Think of an object as a container that holds data and functions related to a specific entity or concept.
  • Use the analogy of a real-world object, like a car, with properties (e.g., color, speed) and methods (e.g., startEngine).

Object Types

TypeScript allows you to define and work with object types.
Object types specify the structure of objects, including the properties they can have and the data types of those properties. Object types are especially useful for type checking and ensuring that objects adhere to a specific structure.

Using custom types

You can define object types using custom types created with 'type' declaration:

    
        type Car = {
            make: string;
            model: string;
            year: number;
        }

        let myCar: Car = {
            make: "Toyota",
            model: "Camry",
            year: 2022
        }

In this case, the Car type specifies the structure of a car object, including make, model, and year properties.

Example:

    
        type Coordinate = {
            x: number;
            y: number;
        }

        const origin: Coordinate = {
            x: 0, 
            y: 0
        }

To access the individual property we can use (dot) notation: 

        const xPosition = origin.x;
        const yPosition = origin.y;

Working with the object

    
        let coord1: Coordinate = {
            x: 1,
            y: 1
        }

    //To modify the properties
        coord1.x = 20;
        coord1.y = 10;


        //here we can see that type aliases are interchangeable and we 
        //can use it with different types of data

        function printCoord(coord: Coordinate){
            console.log(coord.x, coord.y)
        }

        printCoord(coord)
        printCoord({x: 5, y: 7})
