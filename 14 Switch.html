The switch statement

The switch statement is used to evaluate a single expression against multiple possible values or cases. It provides an alternative to a series of if and else if statements when you need to compare one value against many possible values.

Case labels and the 'default' Case

In a switch statement, you define different cases with case labels. If the expression matches a case label, the code within that case is executed. You can also provide a default case, which is executed when none of the cases match. 

    
        let day = "Wednesday";

        switch(day){
            case "Monday":
                console.log("It's the start of the week.")
                break;

            case "Wednesday":
                console.log("It's midweek.");
                break;

            case "Friday":
                console.log("It's before the weekend.")
                break;

            default:
                console.log("What day is today?")
        }

The switch statement is particularly useful when you need to compare a single value against several cases. It makes your code more concise and easier to read compared to a series of if statements.

The 'break' statement

Each case in a switch statement typically ends with a break statement. The break statement is used to exit the switch statement once a case is matched. Without break, the code would continue to execute for subsequent cases.

The 'default' case

The default case in a switch statement is used to provide a code block that will be executed when none of the specified case labels match the value of the expression being evaluated. It serves as a fallback or "catch-all" case, and it's optional. If no matching cases are found, the code within the default case will run.

Here's an example of how the default case is used: 

    
let day = "Sunday";
switch (day) {
  case "Monday":
    console.log("It's the start of the week.");
    break;
  case "Wednesday":
    console.log("It's midweek.");
    break;
  default:
    console.log("It's not Monday or Wednesday.");
}

In this example, since day is "Sunday," it doesn't match any of the defined cases ("Monday" or "Wednesday"). Therefore, the code within the default case is executed, and it prints "It's not Monday or Wednesday."

Exercise: Grade Calculator

Scenario: You are building a grade calculator program. Given a numeric score, you should determine the corresponding letter grade based on the following scale: 

    
        A: 90 or above
        B: 80-89
        C: 70-79
        D: 60-69
        F: Below 60

You should use a switch statement to implement this. If the score doesn't fall into any of the specified ranges, you should display "Invalid score."

Mnemonic Devices:

  • Think of the switch statement as a menu with various options. The single expression is like your selection, and the case labels are the different menu items.
  • The break statement is like finishing your meal and leaving the restaurant. It helps you exit the switch statement when the appropriate case is matched.
