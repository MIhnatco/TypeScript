Functions

Functions are a fundamental building block of any application in TypeScript, as they are in JavaScript. They allow you to build up layers of abstraction, mimicking classes, information hiding, and modules.

Program

In computer science, a program is a set of instructions that a computer follows to perform a specific task or solve a particular problem

When a program runs, it executes a set of steps and those steps are coded by the programmer. Some programs may be simple with only a few steps and others may be complex with many steps. Regardless of how complex a program is, we should be able to organize it. And here come 'functions' in.

Functions

A function is the most basic organizational unit of a program. Functions can perform nay number of actions and they can contain anything. They can have a single boolean expression or they have complex logic for reading programming. Functions allow us to encapsulate logic into a single package. And once that logic is encapsulated within the function, it can be reused anywhere else in the program. This makes it easier to read, to write and to test your code.

For function to do something, it requires some data. To specify what the function works with, we use something called parameters. Functions can have any number of paramters, although it's best to limite thme to a five or less.

To execut the code withing a function, we have to call it.

When calling a function, we need to provide data for the parameters. When data is provided for function, we call the data arguments to the function.

When we call a fuction, both parties can exchange information. The caller provides information using arguments and they call the function and then the function is able to return data back to the caller. It's important to note that when functions are called the correct argumetns must be supplied. If we provide the arguments in the wrong order...then we'll get an error.

The great thing about TypeScript is that all these checks will be happening as you're writing your code, so you'll know rigth away.
