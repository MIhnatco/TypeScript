Arithmetic Operators

TypeScript provides several arithmetic operators for performing basic mathematical operations. Here are the most commonly used ones:

  • Addition (+): Adds two numbers together.
  • Subtraction (-): Subtracts one number from another.
  • Multiplication (*): Multiplies two numbers.
  • Division (/): Divides one number by another.
  • Modulo (%): Returns the remainder of a division operation. 
    
        let x = 10;
        let y = 3;
        let sum = x + y;
        let difference = x - y;
        let product = x * y;

        let quotient = 10 / 5;
        assert.eqaul(quotient, 2)

        let remainder = 10 % 3;
        assert.eqaul(remainder, 1)

        const inf = 1 / 0;
        assert.equal(inf, Infinity)

        const nan = 0 / 0;
        assert(isNane(nan))

        //negation 
        const neg = -remainder;
        assert.eqaul(neg, -1);

        //negation of negation
        const neg2 = -neg;
        assert.eqaul(neg2, 1);

        //exponent
        const exp = 2 ** 3;
        assert.equal(exp, 8);

Combining Arithmetic Operators:

You can combine multiple arithmetic operators in a single expression. TypeScript evaluates expressions from left to right. 

    
        let expression = 10 + 2 * 5; // Multiplication takes precedence, so this equals to 20

Type Coercion in Mathematical Operations:

TypeScript performs type coercion when you mix data types in mathematical operations. For example, when you add a number to a string, TypeScript converts the number to a string and performs string concatenation. 

    
let num = 5;
let str = "10";
let result = num + str; // The result will be "510" (a string), not 15.

These operators are essential for solving a wide range of problems in programming, from simple calculations to complex mathematical algorithms.
