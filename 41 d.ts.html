Type definition files in TypeScript

In TypeScript, the .d.ts files, also known as declaration files, are used to provide type information about JavaScript code that doesn't have TypeScript declarations. These files help TypeScript understand the shape of the existing JavaScript code and provide a better development experience. Writing your own .d.ts files can be beneficial when working with external libraries or modules that lack TypeScript support.

Basic structure of a Declaration file

A declaration file typically has the extension '.d.ts'. The basic structure includes declarations for variables, functions, classes, or modules. 

    
        //example.d.ts 

        //variable declaration
        const myVariable: string;

        //function declaration 
        function myFunction(param: number): vaid;

        //class declaration
        class MyClass {
            constructor(value: string);

            getValue(): string;
        }

Writing Declaratios for existing JavaScript code

When writing a declaration file for existing JavaScript code, you need to declare the types for variables, functions, or classes that are not explicitly typed. 

    
        //JavaScript code
        function sum(x, y, ...args) {
            let total = x + y;
            for (let n of args) {
                total += n;
            }
            return total;
        }

        function maximum(arr) {
            if (arr.length === 0) {
                return null;
            }
            return arr.reduce((a, b) => Math.max(a, b));
        }

        
        function quote(message) {
            return () => { return `"${message}"` }
        }

        function setCase(message, kind) {
            if (kind === "uppercase") {
                return message.toUpperCase();
            } else if (kind === "lowercase") {
                return message.toLowerCase();
            } else {
                throw new Error("invalid kind: must be 'uppercase' or 'lowercase'");
            }
        }

        module.exports = {
            sum, 
            maximum,
            quote,
            setCase
        }

        export function sum(x: number, y: number; numbers: number[]): number;
        export function maximum(arr: number[]): number | null;
        export function quote(message: string): () => string;
