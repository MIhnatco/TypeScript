Arrays

Arrays in TypeScript are ordered collections of values. They can hold values of the same or different data types. TypeScript allows you to specify the type of elements in an array, providing type safety.

Creating Arrays and specifying element types

You can create arrays using square brackets '[]' or the 'Arrray' constructor. You can also specify the type of elements in the array using type annotations. 

    
        //creating an array of numbers
        const numbers: number[] = [1,2,3,4,5]

        //creating an array of strings 
        const colors: string[] = ['red', 'blue', 'orange']

Accessing and modifying Array elements

You can access array elements by their index, and you can modify elements by assigning new values to them. Array indices start from '0'. 

    
        const fruits: string[] = ['apple', 'banana', 'cherry'];
        const firstFruit: string = fruits[0]  //accessing the first element
        fruits[1] = 'grape'  //modifying the second element

Try to avoid this: 

    
        const arr = [1, "home", true]

In TypeScript, it's generally considered a good practice to use arrays to store elements of the same or compatible types. While TypeScript is more permissive than JavaScript and allows arrays to contain mixed types, it's usually better to maintain consistency in your data structures for the following reasons:

  • Type Safety: Using arrays with consistent types ensures that you can catch type-related errors at compile-time. Mixing different types in an array may lead to unexpected runtime errors.
  • Readability: Code that uses arrays with consistent types is typically more readable and easier to understand, as it adheres to a clear and predictable structure.
  • Ease of Maintenance: When you or other developers work with your code in the future, it's simpler to understand and maintain code that follows best practices.

Multidimensional Arrays

Multidimensional arrays are arrays of arrays, allowing you to create a grid or matrix-like data structure. In TypeScript, as in many programming languages, you can use multidimensional arrays to represent data in a structured and organized manner.

Understanding Multidimensional Arrays:

In a one-dimensional array, you have a single list of elements. In contrast, multidimensional arrays have multiple dimensions or "layers" of arrays. The most common form of multidimensional array is the two-dimensional array, which can be thought of as a table with rows and columns.
Here's a simple example of a two-dimensional array in TypeScript: 

    
        const matrix: number[][] = [
          [1, 2, 3],
          [4, 5, 6],
          [7, 8, 9],
        ];

In this example:

  • matrix is a two-dimensional array.
  • It has three rows, each of which is an array.
  • Each row contains three elements, creating a 3x3 grid.

Accessing Elements in Multidimensional Arrays:

To access elements in a multidimensional array, you use multiple indices.
For example, to access the element 5 in the matrix, you would use matrix[1][1], as it's in the second row and second column (zero-based index).

You can provide type annotations for multidimensional arrays just like for one-dimensional arrays. For instance, if you have a two-dimensional array of numbers, you can use 'number[][]' to specify its type.

Multidimensional arrays are a valuable tool for managing structured data and handling grid-like structures. Understanding their usage can be particularly beneficial when dealing with complex data representations.

Arrays provide a lot of methods.

push():

The push() method adds one or more elements to the end of an array and returns the updated length of the array. It modifies the original array. 

    
        const fruits: string[] = ["apple", "banana", "cherry"];
        fruits.push("date");
        console.log(fruits); // Output: ["apple", "banana", "cherry", "date"]

pop():

The pop() method removes the last element from an array and returns that element. It also modifies the original array by removing the last item. 

    
        const numbers: number[] = [1, 2, 3, 4];
        const lastNumber = numbers.pop();
        console.log(lastNumber); // Output: 4
        console.log(numbers); // Output: [1, 2, 3]

splice():

The splice() method allows you to change the contents of an array by removing, replacing, or adding elements. It takes three arguments: the starting index, the number of elements to remove, and optional elements to add. It returns an array containing the removed elements. 

    
        const fruits: string[] = ["apple", "banana", "cherry", "date"];
        const removed = fruits.splice(1, 2, "grape", "kiwi");
        console.log(removed); // Output: ["banana", "cherry"]
        console.log(fruits); // Output: ["apple", "grape", "kiwi", "date"]

forEach():

The forEach() method is used to iterate over each element in an array and execute a provided function for each element. It does not modify the original array; it's often used for performing actions on array elements without changing the array itself. 

    
        const numbers: number[] = [1, 2, 3, 4, 5];
        numbers.forEach((number) => {
          console.log(number * 2);
        });

This will output the doubled values of each element in the array.

These array methods are fundamental for working with arrays in TypeScript and JavaScript. Understanding their use cases and how to apply them effectively can greatly simplify array manipulation in your code.

Arrays offer a way to store and manipulate collections of values of the same type.
They are defined using square brackets and can be populated with values at initialization, or later using various methods such as push(), splice(), and concat().
Arrays can be of a fixed length or dynamically resized as needed, and they can be used with various array methods to perform common operations like sorting, filtering, and mapping.
