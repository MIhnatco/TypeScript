Type

A set of values and the things you can do with them

In TypeScript, every value has a type, and the type determines what operations can be performed on that value. TypeScript supports a wide range of data types, including basic types such as boolean, number, string, null, and undefined, as well as more complex types such as arrays, tuples, enums, and interfaces.

Any

In TypeScript, the any type is a special type that can be used to represent any type of value. It is a dynamic type that allows developers to opt-out of type checking for a particular variable, function, or object. The any type is useful when working with existing JavaScript code that does not have type annotations or when the type of a value is unknown or cannot be determined at compile-time.

TypeScript will not perform any type checking on that variable, which means that it is possible to assign values of any type to that variable. This can lead to unexpected behavior and errors in the code. 

    
        let something: any = "hello, world!";
        something = 23;
        something = true;

In this example:

  • The variable 'something' is initially assigned a string value 'Hello, world!'.
  • Later, it is reassigned with a number (23) and then with a boolean (true).

The any type allows the variable to hold values of any type, and TypeScript does not perform type checking on variables of type any. While this flexibility can be useful in certain scenarios, it's important to use the any type judiciously to maintain type safety and avoid unexpected runtime errors

Type literal

A type that represents a single value and nothing else.

In TypeScript, a type literal is a type that represents a specific value. It is a way to define a type that can only have a single value, rather than a range of values. Type literals are used to create more precise types that can help catch errors at compile-time.

TypeScript supports three types of literal types: string, number, boolen.

For example, a string literal type can be used to represent a specific string value, like this: 

    
        let myString: "Hello world";
        myStr = "Hello world";
        myStr = "World of warcraft"; //Type '"World of warcraft"' is not assignable to type '"Hello world"'.

In this example, the variable myString is declared as a string literal type that can only have the value "Hello World". If we try to assign a different string value to myString, TypeScript will throw a compile-time error.

Type literals can also be used in combination with union types to create more complex types. For example, we can define a type that can only have the values "success" or "error" like this:

    
type Result = "success" | "error";
let myResult: Result = "success"; // OK
myResult = "error"; // OK
myResult = "warning"; // Error: Type '"warning"' is not assignable to type 'Result'

In this example, the Result type is a union of the string literal types "success" and "error". The variable myResult can only have one of these two values.

Boolean Logic

Boolean logic is a fundamental concept that involves evaluating true/false statements. It is a system of logical thought that uses true and false values to make deductions or draw conclusions.
Boolean logic is used to express conditions and make decisions based on the truth or falsehood of these conditions.

Boolean Data Type in TypeScript:

In TypeScript, the boolean data type represents two values: true and false. Booleans are used to express binary logic and are fundamental to decision-making in programming. 

    
        let isRaining: boolean = true;
        let hasPermission: boolean = false;

Logical Operators:

TypeScript provides several logical operators to work with boolean values: 

    
        AND (&&): Returns true if both operands are true.
        OR (||): Returns true if at least one operand is true.
        NOT (!): Negates the value; if the operand is true, it becomes false, and vice versa.

        let sunny = true;
        let warm = true;
        let isSummer = false;
        
        let goingOutside = sunny && warm; // true
        let needACoat = !warm; // false
        let beachDay = sunny && (warm || isSummer); // true

Boolean Expressions:

Boolean expressions are combinations of variables, values, and logical operators that result in a boolean value. They are used in decision-making and control flow in your programs. 

    
        let isWeekend = true;
        let hasHomework = false;
        
        if (isWeekend && !hasHomework) {
          console.log("Let's go have fun!");
        } else {
          console.log("Time to study.");
        }
