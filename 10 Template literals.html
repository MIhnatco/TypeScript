Template Literals

Template literals are a way to create strings that can contain placeholders for dynamic values or expressions. They are defined using backticks (``) instead of single or double quotes. 

    
        const name = "Alice";
        const greeting = `Hello, ${name}!`;

In the example above, ${name} is a placeholder that can be replaced with the value of the name variable.

Interpolating Variables and Expressions:

Template literals allow you to interpolate variables and expressions directly into the string using ${}. This makes it easy to include dynamic content in your strings. 

    
        function greet(message){
            console.log(`Hello, ${message}!`)
        }

We use the 'back quotes', inside we can place a text and then we can use the dynamic value... we use 'dollar sign' with curly braces and then a variable that you want to substitute.

So if we call the function.. 

    
        greet("TypeScript")

and run this program..

    
        Hello, TypeScript!

So this is the simples thing we can do with templates strings.

Multiple Strings:

Template literals also make it simple to create multiline strings without needing to add line breaks explicitly. This is especially useful for formatting longer text. 

    
        const poem = `
            Roses are red, 
            Violets are blue, 
            Sugar is sweet, 
            And so are you.`;

It's also possible to put expressions within a template string. So let's say: 

    
        let kids =  2;
        let adults = 4;
        let totalPeople = `There are ${kids + adults} people`;

        console.log(totalPeople);

Just as with functions, we don't want to put too many things in a template strings, because it's going to be hard to separate the part of string with the part that's being calculated. 

            
                let kids =  2;
                let adults = 4;
                const totalAttending = kids + adults;
                const totalPeople = `There are ${totalAttending} people`;
        
                console.log(totalPeople);

So that's a way you can kind of simplify.

The last thing in this lesson we need to mention..we can actually put template strings into other template strings. 

    
        let kids =  2;
        let adults = 4;
        const totalAttending = kids + adults;
        const message = `There are ${totalAttending} people`;

        console.log(`${message} to seat at the restaurand.`)

This has been composed from two template strings and that's all the interesting parts of the template strings.
They just enable us to easily create strings by composing other strings and pieces of data into one, complete message that we can then use later in our code.

Mnemonic Devices:

  • Think of template literals as strings with superpowers, where you can easily embed variables and expressions.
  • Backticks (`) are your "magic wand" for creating dynamic and multiline strings.

They are especially useful when working with dynamic content, such as user-generated data or when formatting text.
