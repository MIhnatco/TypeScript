Iterators

Iterators are used to traverse collections of data , and the 'for...of' loop provides an elegant way to iterate over elements.

  • Iterators are objects that provide a way to traverse through a collection of data step by step.
  • They allow you to access elements one at a time, making it easy to work with large data sets efficiently.

Using the 'for..of' loop:

  • The 'for..of' loop simplifies the process of iterating over iterable objects, such as arrays, maps, and custom collections.
  • It provides a clean syntax for iterating through the elements.

Example:

    
//Define an array of tasks
const tasks: string[] = ['Buy groceries', "Complete assignment", "Walk the dog"] 

//Use the for...of loop to iterate over the tasks
console.log('To-Do List:')

for (const tasks of tasks){
    console.log(`- ${task}`);
}

In this example:

  1. We have an array called 'tasks' that contains a list of to-do items.
  2. We use the 'for...of' loop to iterate over the 'tasks' array. The 'task' variable in the loop represents each individual task in the array.
  3. Within the loop, we print each task with a bullet point in front of it, creating a simple to-do list.

When you run this code, it will iterate over the 'tasks' array and display each task in the console.

In TypeScript, the for...of loop is typically used with iterable data structures like arrays, maps, sets, and custom iterable objects. Objects are not inherently iterable, so you can't directly use a for...of loop with them.
However, you can use the for...in loop to iterate over the properties of an object. Here's an example: 

    
// Define an object with properties
const person = {
    firstName: "John",
    lastName: "Doe",
    age: 30,
};

// Use a for...in loop to iterate over the object's properties
console.log("Person's Information:");
for (const key in person) {
    if (person.hasOwnProperty(key)) {
        console.log(`${key}: ${person[key]}`);
    }
}

In this example:

  • We have an object named person with properties like firstName, lastName, and age.
  • We use a for...in loop to iterate over the properties of the person object. The loop iterates over the keys (property names) of the object.
  • We use the hasOwnProperty method to ensure that we only access the object's own properties and not properties inherited from the prototype chain.

When you run this code, it will iterate over the properties of the person object and display their names and values. Please note that this is not a typical use case for for...of, as it's more commonly used with iterable data structures like arrays.
